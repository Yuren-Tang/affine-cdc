# 第五基石:交叉配對位元的良定性(T5)

狀態:紙面草案(2026-07-12,待審定封閉)。Lean:同日完成(`DualConfig.lean`)。

## 設定

`Γ` 任意 F₂-向量空間;`W ≤ Γ` 平面;`D = W ∖ {0}` 三方向。

**合法對偶配置空間**(原稿式 (7) 的無排序形式):

$$K(W) := \Big\{\eta : D \to \Gamma^* \;\Big|\; \eta_h(h) = 0 \ (\forall h),\quad \eta_x + \eta_y + \eta_z = 0 \ \big(\forall\, x+y+z = 0\big)\Big\}.$$

註:三方向的「和為零三元組」恰是「兩兩相異三元組」(平面內 $x+y+z=0$ 且皆非零 ⟺ 兩兩相異),故求和條件無需枚舉 $D$——這是規範自由的表述。$K(W)$ 是 $(D \to \Gamma^*)$ 的子空間。

## 定理 D1(交叉配對良定)

對 $\eta \in K(W)$,全部六個交叉值

$$\eta_h(h'),\qquad h \ne h' \in D$$

相等。記其公共值為 $\beta(\eta)$。

**證明**。兩步。

*第二變元穩定*:若 $a, b \ne h$,則 $\eta_h(a) = \eta_h(b)$——當 $a \ne b$ 時 $b = a + h$(平面),故 $\eta_h(b) = \eta_h(a) + \eta_h(h) = \eta_h(a)$。

*換位*:對 $x \ne y$,取第三方向 $z$;在 $z$ 上求值和式 $\eta_x + \eta_y + \eta_z = 0$:

$$0 = \eta_x(z) + \eta_y(z) + \eta_z(z) = \eta_x(y) + \eta_y(x),$$

(第二變元穩定 + $\eta_z(z)=0$),特徵二下即 $\eta_x(y) = \eta_y(x)$。

任兩個交叉對 $(a,b), (c,d)$ 經此兩步(至多三次改寫)相連。∎

## 定理 D2(β 線性)

$\beta : K(W) \to \mathbf F_2$ 是線性泛函(固定任一交叉對求值即得,與 D1 一致)。∎

## 定理 D3(消失刻畫)

$\beta(\eta) = 0$ 當且僅當每個 $\eta_h$ 消滅整個 $W$。

**證明**。$\eta_h$ 在 $W = \{0, h, a, b\}$ 上的值為 $0, 0, \beta, \beta$。∎

**詮釋**(佐 T7):$\beta$ 把原稿的 $\lambda = \eta_b(x)$(依賴排序的座標式)升格為 $K(W)$ 上的內在線性位元;原稿式 (8) 的「只有 $b$ 邊貢獻」是 $\beta$ 在特定規範(localBase)下的表示。$\beta = 0$ 的幾何意義:全部對偶資料落在 $W^\perp$,即在 $Q$-層面全部退化。

## 假設帳目

需要:F₂、$W$ 平面。**不需要**:$\dim\Gamma = 3$、$\Gamma$ 有限、$D$ 排序、補空間。
(對比:原稿的 $\lambda$-計算在 $\Gamma = \mathbf F_2^3$ + 排序下進行;
本定理顯示良定性與三維無關——三維只在 T6 唯一性與 T7 分支恆等式進場。)

## 來源分類

D1–D3 原稿隱含(其式 (7)–(8) 的計算即 D1 特例);無排序表述與「$K(W)$ 是子空間、β 是泛函」的定位為本文新增。
