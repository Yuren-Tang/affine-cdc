# 第四基石:coker δ 的規範分類(T4)

狀態:紙面草案(2026-07-12,待審定封閉)。Lean:待 T3 紙面審定後動工。

## 設定(抽象;圖無關)

`C⁰`, `C¹` 任意 F₂-向量空間(一般域亦可,C1 之外不用特徵二),固定線性映射

$$\delta : C^0 \to C^1.$$

**仿射黏合系統**(linear part 為 δ):方程 $\delta m = c$,由 $c \in C^1$ 決定。
**規範變換**(重選局部原點):$m \mapsto m + a$,對應 $c \mapsto c + \delta a$。
記 $c \sim c' \iff \exists a,\; c' = c + \delta a$。

## 定理 C1(規範分類;完整不變量)

商映射 $c \mapsto [c] \in \operatorname{coker}\delta$ 誘導雙射

$$C^1/\sim \;\xrightarrow{\ \sim\ }\; \operatorname{coker}\delta,$$

且:

1. $c \sim c' \iff [c] = [c']$(**完整性**:任何規範不變量必經 $[c]$ 分解);
2. $[c] = 0 \iff \delta m = c$ 有解(**障礙**);
3. 若 $[c] = [c']$,則兩系統恰差一次局部原點重選(等價的「幾何」形式)。

**證明**。1、2 是 coker 的定義展開:$[c]=[c'] \iff c - c' \in \operatorname{im}\delta \iff \exists a,\, c' = c + \delta a$;$[c]=0 \iff c \in \operatorname{im}\delta$。3 即 1 的重述。雙射性由 1(單)與商映射滿性(滿)。∎

**詮釋**:「canonical obstruction」不是修辭:$[c_f]$ 是局部座標平移下的**完整**
不變量——任何不變於重選局部原點、且能分類此仿射系統的量,都因 1 而經由 $[c]$。

## 定理 C2(對偶判準)

$c \in \operatorname{im}\delta$ 當且僅當:對每個線性泛函 $\eta : C^1 \to \mathbf F_2$
滿足 $\eta \circ \delta = 0$,皆有 $\eta(c) = 0$。

**證明**。⟹ 顯然。⟸:$\operatorname{im}\delta$ 是子空間,任何子空間是包含它的
超平面核之交(取補空間;一般維數用選擇公理,有限維初等)。若 $c \notin \operatorname{im}\delta$,
取 $\eta$ 消滅 $\operatorname{im}\delta$ 而 $\eta(c) = 1$。∎

**備註**:原文只在有限維使用($\Gamma^{E}$ 有限);本定理在任意維成立,
但 T7 只需有限情形。Lean 中有限維可用 `Module.Dual` +
`Submodule.exists_dual_map_eq_bot_of_notMem` 一類引理(動工時查名)。

## 特化介面(C3;圖層,佐 T7)

三次圖 $G$、nowhere-zero flow $f : E \to \Gamma$、頂點平面
$W_v = \{0, f(e_1), f(e_2), f(e_3)\}$($e_i \ni v$),取

$$C^0 = \Gamma^{V},\qquad C^1 = \bigoplus_{e} Q_e,\qquad Q_e = \Gamma/\langle f(e)\rangle,$$

$$(\delta_f m)_e = [m_u + m_v],\qquad c_f(e) = \kappa_{u,e} + \kappa_{v,e},$$

其中 $\kappa_{v,e} := \kappa_{f(e)}(W_v)$(T3 B3 的唯一非零元素)。

**命題 C3(黏合方程)**。以刪去點 $(m_v)_{v}$ 給出的各頂點局部偶族
(T1)在邊 $e = uv$ 上黏合(即兩端指派同一條線)當且僅當

$$(\delta_f m)_e = c_f(e).$$

**證明**。兩端在 $Q_e$ 中的線參數(T3 B4)為 $[m_u] + \kappa_{u,e}$ 與
$[m_v] + \kappa_{v,e}$;由 B1(線 = 陪集 = $Q_e$ 元素)相等即黏合。特徵二移項。∎

由是,**Fano 相容性定理**(T7 的目標)即 $[c_f] = 0$ 對一切三次圖 + NWZ flow;
其否定的分類空間(一般 Γ 上何時 $[c_f] \ne 0$)= 障礙理論分支。

## 與原稿/原庫的對應(佐 comparison layer)

原稿的 $d_e = g_{u,e} + g_{v,e}$(依 `localBase`/排序)所定義的系統,經 B2
投影後與 $c_f$ 系統**同規範類**:localBase 的局部族由 T1 有唯一刪去點 $m_v^0$,
其標籤 $= [m_v^0] + \kappa_{v,e}$,故 $[d_e] = (\delta_f m^0)_e + c_f(e)$,即
$[d] \sim c_f$。這把「原稿的排序選擇只改變規範代表」變成一行計算
(comparison layer 的核心引理,屆時落地)。

## 假設帳目

- C1:任意域、任意維;C2:任意域(無限維用選擇);C3:F₂ + T1/T3 介面。
- 全部不需要 $\dim\Gamma = 3$、不需要 flow 的存在性(只用其值)。
