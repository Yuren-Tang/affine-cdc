# 第七基石:分支恆等式與全局消失(T7)

狀態:紙面草案(2026-07-13,待審定封閉)。Lean:待 T6/T7 審定後動工(圖層 + 收官)。

**這是 `dim Γ = 3` 唯一真正進場的地方,且是精確刻畫(F1)。**

## F0:圖層(dart 形式化;規範自由的組合結構)

一個**三次 dart 結構**是資料 $(Δ, \sigma, \mathsf v)$:

- 有限集 $Δ$(darts / 半邊);
- 無不動點對合 $\sigma : Δ \to Δ$(邊配對;無環三次多重圖恰為此形,平行邊自然容許);
- 映射 $\mathsf v : Δ \to V$,每根纖維恰 3 個元素(頂點,三次)。

邊集 $E := Δ/\sigma$。**注意**:原庫的 `endAt e 0/1`(端點編號)與
`edgeAt v i`(頂點槽位)都是此結構上的座標;dart 形式沒有這些選擇。

**流**:$f : Δ \to \Gamma$,滿足 $f(\sigma d) = f(d)$(特徵二下無向)與
$\sum_{d \in \mathsf v^{-1}(u)} f(d) = 0$(每頂點);nowhere-zero:$f(d) \ne 0$。

**頂點平面**:$W_u := \{0, f(d_1), f(d_2), f(d_3)\}$($\mathsf v^{-1}(u) = \{d_1,d_2,d_3\}$)。
NWZ + 和零 ⟹ 三值兩兩相異非零 ⟹ `IsPlane W_u`,且 $d \mapsto f(d)$ 是
$\mathsf v^{-1}(u) \xrightarrow{\sim} D_u := W_u∖\{0\}$ 的雙射。

**邊商與天然類**(T3 介面):$Q_d := \Gamma/\langle f(d)\rangle$;因
$f(\sigma d) = f(d)$,$Q_{\sigma d} = Q_d$,故 $Q_e$ 對邊良定——dart 形式下
這是**定義相等**,無需引理。$\kappa_d := \kappa_{f(d)}(W_{\mathsf v(d)}) \in Q_d$
(T3 B3);

$$c_f(e) := \kappa_d + \kappa_{\sigma d} \in Q_e,\qquad
(\delta_f m)_e := [m_{\mathsf v(d)} + m_{\mathsf v(\sigma d)}]$$

($\sigma$-對稱,故對邊良定)。黏合方程 $(\delta_f m) = c_f$ 即 T4 C3。

## 定理 F1(分支恆等式;dim 3 的精確刻畫)

設 `W ≤ Γ` 平面,$\eta \in K(W)$。

1. 若 $\operatorname{codim} W = 1$(即 $\dim\Gamma = 3$),則

$$\beta(\eta) \;=\; \#\{h \in D : \eta_h \ne 0\} \pmod 2. \tag{F1}$$

2. 反之,若 $\operatorname{codim} W \ge 2$,則 (F1) 有反例。

**證明**。(1) 若 $\beta = 1$:每個 $\eta_h$ 在另兩方向取 1,故全非零,計數 3 ≡ 1 ✓。
若 $\beta = 0$:全部 $\eta_h$ 消滅 $W$(T5 D3),即 $\eta_h \in W^\perp \cong (\Gamma/W)^*$;
codim 1 下 $W^\perp = \{0, \theta\}$ 恰有一個非零元素,故每個 $\eta_h \in \{0,\theta\}$;
和式 $\eta_x + \eta_y + \eta_z = 0$ ⟹ 非零者數目為偶(0 或 2)✓。

(2) codim ≥ 2:取線性無關 $\mu, \mu' \in (\Gamma/W)^*$,拉回為消滅 $W$ 的泛函;
令 $\eta_x = \mu,\ \eta_y = \mu',\ \eta_z = \mu + \mu'$:合法($\eta_h(h) = 0$ ✓、
和零 ✓)、$\beta = 0$、但非零者 3 個,奇偶 = 1 ≠ 0。∎

**詮釋**:E3(任意維唯一性)說 β 是唯一的不變位元;F1 說它的「支集計數表示」
恰在 codim 1 可用。全局論證(F2)需要的正是計數表示(每邊兩端),故
$\Gamma = \mathbf F_2^3$ 對此證明機制不可替換——這是「為甚麼偏偏是八流」的
定理級答案。

## 定理 F2(全局消失 / Fano 相容性定理)

$\Gamma = \mathbf F_2^3$,$(Δ,\sigma,\mathsf v)$ 三次 dart 結構,$f$ NWZ 流。則

$$c_f \in \operatorname{im}\delta_f,\qquad\text{即}\qquad [c_f] = 0 \in \operatorname{coker}\delta_f.$$

**證明**。由 T4 C2(對偶判準)只需:每個消滅 $\operatorname{im}\delta_f$ 的
$\eta = (\eta_e) \in \prod_e Q_e^*$ 消滅 $c_f$。

*對偶條件的翻譯*:$Q_e^* \cong \{\phi \in \Gamma^* : \phi(f(e)) = 0\}$(T3 B2 的
對偶形式);$\eta$ 消滅 $\operatorname{im}\delta_f$ ⟺ 每頂點
$\sum_{d \in \mathsf v^{-1}(u)} \eta_{e(d)} = 0$。故每頂點的資料
$(\eta_{e(d)})_{d}$ 經 $d \mapsto f(d)$ 是 $K(W_u)$ 的元素(兩公理逐條對應)。

*逐頂點展開*:$\eta_{e}(c_f(e)) = \eta_e(\kappa_d) + \eta_e(\kappa_{\sigma d})$,故

$$\langle\eta, c_f\rangle = \sum_e \eta_e(c_f(e)) = \sum_u \sum_{d \in \mathsf v^{-1}(u)} \eta_{e(d)}(\kappa_d).$$

*局部求值*:$\kappa_d$ 是「另兩方向」的公共類,故 $\eta_{e(d)}(\kappa_d)$ 是交叉值
$= \beta_u$(T5 D1;每頂點三項,$3\beta_u \equiv \beta_u$)。

*計數收束*:由 F1,$\beta_u = \#\{d \in \mathsf v^{-1}(u) : \eta_{e(d)} \ne 0\} \bmod 2$;求和

$$\langle\eta, c_f\rangle = \sum_u \#\{d : \eta \ne 0\} = 2\,\#\{e : \eta_e \ne 0\} = 0 \pmod 2. \qquad\blacksquare$$

## 推論 F3(全局偶族;CDC 介面)

$[c_f] = 0$ ⟹(T4 C1)黏合方程有解 $m$ ⟹(T4 C3 + T1)各頂點局部偶族
$\Phi_{W_u}(m_u)$ 沿全部邊黏合,給出全局邊標籤 $P : E \to$ 二點集,使每頂點
每元素之覆蓋數 ∈ {0, 2}。

由原稿 Lemma 2.1(每 $s \in \Gamma$ 的支集 $M_s$ 是不交圈之並、每邊屬於恰兩個
$M_s$)即得 CDC——依決策記錄,此最後一步為論文 remark / 可選橋接,不入本庫
Lean 終點($[c_f] = 0$ 即終點)。
