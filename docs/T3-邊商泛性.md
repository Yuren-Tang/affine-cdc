# 第三基石:邊商泛性(T3)

狀態:紙面草案(2026-07-12,待審定封閉)。Lean:**B1–B4 全部完成**(`EdgeQuotient.lean`,零 `sorry`,公理乾淨)——形式化先行降低了審定風險;若審定時修改陳述,同步回改 Lean。

## 設定

`Γ` 任意 F₂-向量空間;`h ∈ Γ`,`h ≠ 0`。記商映射

$$\pi_h : \Gamma \to \Gamma/\langle h\rangle =: Q_h.$$

特徵二下 $\langle h\rangle = \{0, h\}$。

---

## 定理 B1(二點集 = 陪集)

方向 $h$ 的二點集恰是 $\langle h\rangle$-陪集:對任意 $p$,

$$\{p,\ p+h\} = p + \langle h\rangle,$$

且 $p \mapsto \{p, p+h\}$ 誘導雙射

$$Q_h \;\xrightarrow{\ \sim\ }\; \{\text{方向 } h \text{ 的仿射線}\}.$$

**證明**。$\langle h\rangle = \{0,h\}$,故 $p+\langle h\rangle = \{p, p+h\}$。兩線相等 $\{p,p+h\} = \{p',p'+h\}$ ⟺ $p - p' \in \{0,h\}$ ⟺ $\pi_h(p) = \pi_h(p')$。∎

*Lean*:`coset_eq_pair`(已證,`OppositeFiber.lean`)。這條使「線」可以在形式化中直接以 $Q_h$ 的元素為原語——Block 1 的 Lean 已按此實現並反向驗證了本定理。

## 定理 B2(邊商泛性;「忘掉基點」的必然性)

$\pi_h$ 在消滅 $h$ 的線性映射中是**始對象**:對任意 F₂-向量空間 $T$ 與線性映射 $q:\Gamma\to T$ 滿足 $q(h)=0$,存在唯一線性 $\bar q: Q_h \to T$ 使 $q = \bar q \circ \pi_h$。

因此,任何「線的參數化」$\lambda$(即對每個基點 $p$ 給出參數 $\lambda(p)$),只要

1. $\lambda$ 對基點線性;
2. $\lambda$ 不分辨同一條線的兩個基點($\lambda(p) = \lambda(p+h)$),

就唯一地經由 $Q_h$ 分解:$\lambda = \bar\lambda\circ\pi_h$。

**證明**。條件 2 等價於 $\lambda(h)=0$(線性 + $\lambda(p+h)=\lambda(p)$)。餘為商空間標準泛性。∎

**詮釋**:$Q_e$ 不是一個方便的選項。一旦要求把同一二點集的兩個基點視為同一對象,$Q_e$ 就是必然出現的最通用參數空間。原稿中的 $\epsilon_e \in \mathbf F_2$ 正是此商等式的提升變數(lifting artifact)。

*Lean*:對應 `Submodule.liftQ` + `Submodule.liftQ_unique`(Mathlib 現成);薄切片只需以我們的 `LineSpace` 語言包裝並陳述唯一性。

## 定理 B3(κ 的唯一性)

設 $W\le\Gamma$ 為平面(`IsPlane`),$h\in W\setminus\{0\}$。則:

1. $\pi_h(W) = W/\langle h\rangle \le Q_h$ 是一維子空間(≅ F₂,恰有一個非零元素);
2. 其唯一非零元素恰為 $\kappa_h$(即另外兩個方向的公共類)。

**證明**。$W = \{0, h, a, a+h\}$($a$ 為任一其他方向)。$\pi_h(W) = \{0, \pi_h(a)\}$,而 $\pi_h(a)\ne0$ 因 $a\notin\{0,h\}$。故 $\pi_h(W)$ 恰有兩個元素,一維。$\kappa_h := \pi_h(a)$ 依定義(選擇無關性已在 T1 證明)。設 $x\in\pi_h(W)$,$x\ne0$,則 $x = \pi_h(a) = \kappa_h$。∎

**與原稿的對應**:原稿第五節「$W$ 的消滅子一維、只有一個非零泛函」是本定理的對偶形式。本定理是原始形式:不需 $\dim\Gamma=3$,不需對偶。

*Lean 目標*:`kappa_mem_image`、`kappa_unique`(`∀ x ∈ (W.map (lineMk h)), x ≠ 0 → x = hP.kappa h` 之形)。

## 定理 B4(局部標籤公式)

在 T1 的分類下,對每個方向 $h$,局部偶族 $\Phi_W(m)$ 的第 $h$ 條線在 $Q_h$ 中的參數為

$$[\Phi_W(m)]_h = [m] + \kappa_h.$$

**證明**。這在我們的表述中即 `localFamily` 的定義;其內容性陳述為:以刪去點 $m$ 分類的局部偶族,其線參數被迫取此仿射形式(由 T1 雙射 + B3 的 $\kappa$ 唯一性)。∎

**向 T4 的介面**:在三次圖頂點 $v$、鄰邊 $e$ 上取 $\Gamma$ 為 flow 值域、$W_v$ 為該頂點兩鄰邊流值張成的平面、$h = f(e)$,則 $\kappa_{v,e} := \kappa_{f(e)}(W_v)$、局部標籤 $= [m_v] + \kappa_{v,e}$,而

$$c_f(e) = \kappa_{u,e} + \kappa_{v,e},\qquad \delta_f m\,(e) = [m_u + m_v]$$

的黏合方程 $\delta_f m = c_f$ 恰是「兩端標籤相等」。T4 在 $\operatorname{coker}\delta_f$ 中做規範分類。
