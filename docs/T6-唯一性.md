# 第六基石:不變線性泛函的唯一性(T6)

狀態:紙面草案 v2(2026-07-13,待審定封閉)。Lean:待審定後動工。

**v2 藍圖變更**:原 v1 只對 `dim Γ = 3` 陳述;現升格為**任意 F₂-向量空間**的
定理,統一剪切證明(v1 的座標收束成為 `C* = F₂` 特例)。由此 **T1–T6 全部
與維數無關;`dim Γ = 3` 的唯一進場點後移至 T7**,且在彼處是精確刻畫(見 T7 F1)。

## 設定

`Γ` **任意** F₂-向量空間;`W ≤ Γ` 平面;`K(W)`、`β` 如 T5。

$$H := \operatorname{Aut}(\Gamma, W),\qquad (T\cdot\eta)_h := \eta_{T^{-1}h}\circ T^{-1}.$$

固定 `W` 的任一補空間 `C`(`Γ = W ⊕ C`;存在性用選擇)。**剪切**:對線性
`s : C → W`,令 `S_s := id_W ⊕ (c ↦ c + s(c))`;則 `S_s ∈ H`、`S_s` 逐點固定
`W`、特徵二下 `S_s^{-1} = S_s`。剪切全體構成 `H` 的子群 `Sh(C) ≅ Hom(C, W)`。

## 定理 E1(K(W) 的結構;任意 Γ)

取兩個相異方向 `x, y`。映射

$$\eta \longmapsto \big(\beta(\eta),\ \eta_x|_C,\ \eta_y|_C\big)$$

是線性同構 $K(W) \cong \mathbf F_2 \times C^* \times C^*$。

**證明**。$\eta_h|_W$ 由 $\beta$ 決定(T5 D3 值表:$0,0,\beta,\beta$);
$\eta_z|_C = \eta_x|_C + \eta_y|_C$ 由和式;故映射單。反之任意
$(b, \mu_1, \mu_2)$ 依值表與和式定義的族滿足 $K(W)$ 兩公理(逐條驗證),故滿。∎

(無選擇形式:正合列 $0 \to K_0(W) \to K(W) \xrightarrow{\beta} \mathbf F_2 \to 0$,
$K_0 = \{\eta : \eta_h|_W = 0\} \cong \{(\mu_h)\in((\Gamma/W)^*)^D : \sum\mu = 0\}$。)

## 定理 E2(β 的 H-不變性)

$\beta(T\cdot\eta) = \beta(\eta)$ 對一切 $T \in H$。

**證明**。$(T\cdot\eta)_h(h') = \eta_{T^{-1}h}(T^{-1}h')$ 是 $\eta$ 在一對相異
方向上的交叉值,由 T5 D1 等於 $\beta(\eta)$。∎

## 定理 E3(唯一性;任意 Γ,只需剪切不變)

設線性泛函 $\ell : K(W) \to \mathbf F_2$ 對剪切子群 $Sh(C)$ 不變。則

$$\ell \in \{0,\ \beta\}.$$

特別地 $\big(K(W)^*\big)^H = \{0, \beta\}$(因 $Sh(C) \le H$ 且 β 不變由 E2)。

**證明**。以 E1 座標寫 $\ell(b, \mu_1, \mu_2) = c\,b + \varphi(\mu_1) + \psi(\mu_2)$,
其中 $\varphi, \psi \in C^{**}$(線性性自動給出此分解)。

*剪切的座標作用*:$(S_s\cdot\eta)_h(c) = \eta_h(c + s(c)) = \mu_h(c) + b\,\lambda_h(s(c))$,
其中 $\lambda_h \in W^*$ 是 T3 B3 的商泛函($\{0,h\}\mapsto 0$、其餘 $\mapsto 1$)。故

$$(b, \mu_1, \mu_2) \longmapsto \big(b,\ \mu_1 + b\,(\lambda_x\circ s),\ \mu_2 + b\,(\lambda_y\circ s)\big).$$

*關鍵同構*:$(\lambda_x, \lambda_y)$ 是 $W^*$ 的基(核相異),故

$$\operatorname{Hom}(C, W) \xrightarrow{\ \sim\ } C^* \times C^*,\qquad s \longmapsto (\lambda_x\circ s,\ \lambda_y\circ s).$$

*收束*:在 $b = 1$ 的配置上,不變性給出對一切 $s$:
$\varphi(\lambda_x\circ s) + \psi(\lambda_y\circ s) = 0$。由關鍵同構,兩個變元
**獨立**取遍 $C^*$;分別取 $(\alpha, 0)$ 與 $(0, \alpha)$ 得 $\varphi = \psi = 0$。
故 $\ell = c\,\beta$。∎

**與 v1 的關係**:$\dim\Gamma = 3$ ⟺ $C^* = \mathbf F_2$,此時關鍵同構即
「$(b,a_1,a_2) \mapsto (b, a_1 + b\lambda_x(w), a_2+b\lambda_y(w))$、由
$\lambda$ 無關性收束」的 v1 論證。一般證明反而更短:無維數計數、無 GL(Γ/W)、
無分情形。

**詮釋**:β 是任意維數下合法對偶配置上唯一的剪切不變線性位元;T7 的分支恆等式
給它的「計數表示」則**只**在 $\dim\Gamma = 3$ 成立(精確刻畫,見 T7 F1)。
「為甚麼偏偏是這條公式」與「為甚麼偏偏是 $\mathbf F_2^3$」在此徹底分離:
前者是任意維的唯一性,後者是 T7 的計數現象。
